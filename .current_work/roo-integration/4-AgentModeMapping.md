## Background: How Roo Custom Modes Work

Roo custom modes are defined in YAML format and can be stored:
1. **Globally** — `C:/Users/{user}/AppData/Roaming/Code/User/globalStorage/rooveterinaryinc.roo-cline/settings/custom_modes.yaml`
2. **Per-project** — `.roo/custom_modes.yaml` at the project root (Roo reads this automatically)

The mode YAML structure (from the currently visible `custom_modes.yaml`):
```yaml
customModes:
  - slug: gsd-executor
    name: GSD Executor
    roleDefinition: |
      [agent system prompt here]
    groups:
      - read
      - edit
      - command
      - mcp
    source: global
```

The `groups` field controls tool access:
- `read` — file reading tools
- `edit` — file writing/editing tools  
- `command` — shell command execution
- `mcp` — MCP tool access
- `browser` — web browsing (if available)

## Agent File Conversion

Agent markdown files (`agents/gsd-executor.md`, etc.) need the same conversion as command files:

### 1. Path Replacement
Same `~/.claude/` → `~/.roo/` replacement as Phase 3.

### 2. Tool Name References in Agent Bodies
Agent markdown files reference tool names in prose and bash snippets. The path substitution handles the `node "$HOME/.claude/..."` references. The prose tool names (Read, Write, Edit, Bash, Task, etc.) in agent role definitions are understood contextually by Roo's AI — **do not mechanically replace these** in prose since the AI interprets the intent.

However, in any `<tools>` frontmatter field in agent files, replace:
```yaml
tools: Read, Write, Edit, Bash, Grep, Glob
```
→ Remove this field entirely (tool access is controlled by mode `groups` in Roo)

### 3. CLAUDE.md → Roo Rules Reference
In `gsd-executor.md` and `gsd-phase-researcher.md`, the agent checks for `./CLAUDE.md`:
```
**Project instructions:** Read `./CLAUDE.md` if it exists
```
Update to check both:
```
**Project instructions:** Read `./CLAUDE.md` or `.roo/rules/*.md` files if they exist in the working directory.
```

### 4. Skills Directory Reference
```
`.claude/skills/` or `.agents/skills/`
```
→ 
```
`.roo/skills/`, `.claude/skills/`, or `.agents/skills/`
```

## `installRooModes()` Installer Function

```js
function installRooModes(rooConfigDir) {
  const yaml = require('js-yaml'); // or use simple string templating if yaml not available
  const agentsDir = path.join(__dirname, '..', 'agents');
  const modesEntries = [];
  
  const agentModeMap = {
    'gsd-executor.md':           { slug: 'gsd-executor',           name: 'GSD Executor',           groups: ['read','edit','command','mcp'], whenToUse: 'Executes GSD PLAN.md files atomically' },
    'gsd-planner.md':            { slug: 'gsd-planner',            name: 'GSD Planner',            groups: ['read','edit','command','mcp'], whenToUse: 'Creates GSD PLAN.md files' },
    'gsd-phase-researcher.md':   { slug: 'gsd-phase-researcher',   name: 'GSD Phase Researcher',   groups: ['read','edit','command','mcp'], whenToUse: 'Researches domain before phase planning' },
    'gsd-project-researcher.md': { slug: 'gsd-project-researcher', name: 'GSD Project Researcher', groups: ['read','edit','command','mcp'], whenToUse: 'Researches project domain during new-project' },
    'gsd-research-synthesizer.md': { slug: 'gsd-research-synthesizer', name: 'GSD Research Synthesizer', groups: ['read','edit'], whenToUse: 'Synthesizes research findings' },
    'gsd-codebase-mapper.md':    { slug: 'gsd-codebase-mapper',    name: 'GSD Codebase Mapper',    groups: ['read','edit','command'], whenToUse: 'Maps existing codebases' },
    'gsd-roadmapper.md':         { slug: 'gsd-roadmapper',         name: 'GSD Roadmapper',         groups: ['read','edit'], whenToUse: 'Generates ROADMAP.md' },
    'gsd-debugger.md':           { slug: 'gsd-debugger',           name: 'GSD Debugger',           groups: ['read','edit','command'], whenToUse: 'Systematic debugging' },
    'gsd-verifier.md':           { slug: 'gsd-verifier',           name: 'GSD Verifier',           groups: ['read','command'], whenToUse: 'Post-execution verification' },
    'gsd-plan-checker.md':       { slug: 'gsd-plan-checker',       name: 'GSD Plan Checker',       groups: ['read'], whenToUse: 'Verifies plan quality' },
    'gsd-integration-checker.md':{ slug: 'gsd-integration-checker',name: 'GSD Integration Checker',groups: ['read','command'], whenToUse: 'Milestone integration validation' },
  };
  
  for (const [file, meta] of Object.entries(agentModeMap)) {
    const agentPath = path.join(agentsDir, file);
    if (!fs.existsSync(agentPath)) continue;
    
    let roleDefinition = fs.readFileSync(agentPath, 'utf8');
    roleDefinition = replacePathsForRoo(roleDefinition);
    roleDefinition = updateProjectInstructionsForRoo(roleDefinition);
    
    modesEntries.push({
      slug: meta.slug,
      name: meta.name,
      roleDefinition,
      whenToUse: meta.whenToUse,
      groups: meta.groups,
      source: 'global'
    });
  }
  
  // Write modes to ~/.roo/get-shit-done/roo-modes-generated.yaml
  const modesPath = path.join(rooConfigDir, 'get-shit-done', 'roo-modes-generated.yaml');
  const modesYaml = formatModesAsYaml(modesEntries);
  fs.writeFileSync(modesPath, modesYaml);
  
  console.log('\n  GSD modes generated at: ' + modesPath);
  console.log('  To register modes in Roo, add this to your Roo custom_modes.yaml:');
  console.log('  (See: ~/.roo/get-shit-done/roo-modes-generated.yaml)');
  
  // Attempt to merge into global custom_modes.yaml if it can be found
  const globalModesPath = findRooGlobalModesPath();
  if (globalModesPath) {
    mergeRooModes(globalModesPath, modesEntries);
    console.log('  ✓ Modes registered in: ' + globalModesPath);
  } else {
    console.log('  ℹ Could not locate global Roo settings. Please manually merge roo-modes-generated.yaml.');
  }
}

function findRooGlobalModesPath() {
  // Try known OS-specific paths
  const candidates = [
    // Windows
    path.join(os.homedir(), 'AppData', 'Roaming', 'Code', 'User', 'globalStorage', 
              'rooveterinaryinc.roo-cline', 'settings', 'custom_modes.yaml'),
    // macOS
    path.join(os.homedir(), 'Library', 'Application Support', 'Code', 'User', 
              'globalStorage', 'rooveterinaryinc.roo-cline', 'settings', 'custom_modes.yaml'),
    // Linux
    path.join(os.homedir(), '.config', 'Code', 'User', 'globalStorage',
              'rooveterinaryinc.roo-cline', 'settings', 'custom_modes.yaml'),
  ];
  
  return candidates.find(p => fs.existsSync(p)) || null;
}
```
